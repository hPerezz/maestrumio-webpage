Beta Update

Maestrum v1.0.5

Released January 2025

Hi,

This update is specifically for our beta users. Good news: this is the last time you’ll need to update manually (delete the old .app and replace it with the new one).

From now on, updates will happen automatically thanks to auto-update and code signing.

Also important: performance has improved a lot in this version. Tracking feels noticeably faster, smoother, and more responsive overall.

Ready to upgrade?

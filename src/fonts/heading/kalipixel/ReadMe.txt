Kalipixel Typeface

Designed by Pengedar Seni

Thank you for purchasing Kalipixel! Below are instructions to install and use the font properly.

Installation Guide:

Windows:

Unzip the downloaded file.

Right-click on the Kalipixel.otf or Kalipixel.ttf file and select Install.

The font is now ready to use in any software.

MacOS:

Unzip the downloaded file.

Double-click the Kalipixel.otf or Kalipixel.ttf file.

Click Install Font in Font Book.

The font is now available in your applications.

Webfont Usage:

Use WOFF/WOFF2 for web embedding.

Add the following CSS rule:

@font-face {
  font-family: 'Kalipixel';
  src: url('Kalipixel.woff2') format('woff2'),
       url('Kalipixel.woff') format('woff');
  font-weight: normal;
  font-style: normal;
}
